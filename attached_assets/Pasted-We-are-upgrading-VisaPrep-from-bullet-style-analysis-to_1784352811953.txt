2. ADD DOCUMENT VALIDATION BEHAVIOR

Update the SYSTEM_PROMPT so the model first determines whether the extracted document contains enough recognizable DS-160 application information.

If it is likely a completed DS-160:

"documentAssessment": {
  "isLikelyDS160": true,
  "message": null
}

If it is not a DS-160, or contains too little recognizable DS-160 information:

"documentAssessment": {
  "isLikelyDS160": false,
  "message": "We couldn't identify enough DS-160 application information in this document to generate personalized interview preparation. Please upload your completed DS-160 application PDF."
}

For an invalid or insufficient document:
- Do not pretend it is a DS-160.
- Do not generate generic preparation advice.
- Set applicationProfile to the same helpful message.
- Set submissionDate to null.
- Return empty arrays for topPreparationAreas, sections, crossSectionObservations, and readyToExplain.

3. REFINE THE AI OUTPUT INSTRUCTIONS

Update Stage 2 so the model understands that each structured item has a distinct purpose:

observation:
State only what the submitted application indicates.

whyItMayComeUp / whyItMatters:
Translate the application detail into interview understanding. Explain why a consular officer may naturally seek clarification or context. Do not imply certainty or predict the outcome.

whatToBeReadyToExplain / preparationGuidance:
Place responsibility on the applicant. Explain what they should review and prepare to discuss honestly, clearly, and in their own words.

Use calm, educational language.

Never merely repeat the same sentence across multiple fields.

Never create fear.

Never describe common facts such as previous refusals, traveling alone, relatives in the United States, sponsors, employment changes, or multi-city travel as inherently problematic.

Example of the required quality:

{
  "title": "Previous Visa Refusal",
  "observation": "Your application indicates that you were refused a U.S. visa in 2022.",
  "whyItMayComeUp": "Previous refusals are part of many applicants' histories, and a consular officer may want to understand the circumstances of the earlier application and what has changed since then.",
  "whatToBeReadyToExplain": "Be prepared to explain the earlier application honestly and describe any significant changes in your employment, travel purpose, finances, or personal circumstances since 2022."
}

This example is a writing standard, not content to copy unless supported by the uploaded application.
