# VisaPrep

VisaPrep helps visa applicants understand the DS-160 application they submitted and prepare to communicate clearly, confidently, and honestly during their interview.

## Current Prototype

The prototype currently includes:

- VisaPrep landing screen
- DS-160 PDF upload interface
- Processing and analysis states
- Application Insights results interface
- API route for document analysis
- Early PDF extraction integration

## Magic Moment

A user uploads their submitted DS-160 and receives meaningful Application Insights that help them better understand their own application.

## Product Principles

VisaPrep is built around:

- Preparation
- Clarity
- Confidence
- Truthful communication
- Ethical interview preparation

VisaPrep does not guarantee visa approval, predict outcomes, or help users misrepresent their circumstances.
